# Word Search Game 🔍

An interactive, progressive word search game built with vanilla JavaScript.

## Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎮 Progressive difficulty levels
- 💾 Automatic progress saving
- ✨ Smooth animations and touch support
- 🎯 Word highlighting and validation

## How to Play

1. Open `index.html` in your browser
2. Find all words from the list
3. Select words by clicking/touching the first and last letter
4. Complete each level to unlock the next

## Tech Stack

- HTML5
- CSS3 (Grid + Flexbox)
- Vanilla JavaScript (ES6+)

## Live Demo

Simply open `index.html` in any modern browser - no build process required!
